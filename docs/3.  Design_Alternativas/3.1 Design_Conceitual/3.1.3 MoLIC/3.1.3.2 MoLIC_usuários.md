## Login e Diagnóstico Diário
- Usuário realiza Login e é direcionado para uma área onde é verificado se é necessário habilitar as opções de acessibilade para uso especial (Usuário PcD). Logo após essa análise, tanto Usuário Comum quanto Usário PcD são encaminhados para o Diagnóstico Diário, onde serão coletadas informações sobre como estão se sentindo.

## Músicas
- O Usuário Comum poderá escolher músicas para relaxar. O Usuário PcD não possuirá acesso a essa funcionalidade.

## Mensagem Anônima
- O Usuário Comum poderá enviar mensagens anônimas para outros usuários sem ser identificado, sendo elas mensagens de apoio emocional ou, simplemenste, um emoji feliz. O Usuário PcD não poderá mandar mensagens anônimas, apenas recebê-las.
<br>
<p align="center"> <img src="https://github.com/ailinha01/IHC-TRABALHO/blob/4f38bee6064023789184688d7bb4ad366f36eb19/docs/Imagens/Imagens_MoLIC/diagramaMOLIC_mensagem_an%C3%B4nima.drawio.png" alt="" width="500" /></p>
<br>

## Área Desabafo
- O Usuário Comum terá acesso a uma Área do Desabafo que funcionará como um diário, permitindo anotar suas frustações e até mesmo acontecimentos do cotidiano. O Usuário PcD não terá acesso a essa funcionalidade.
<br>

## Ombro Amigo
- O Usuário Comum terá acesso a funcinalidade Ombro Amigo, que permitirá acionar um amigo vinculado para uma simples interação, como pedidos de ajuda ou uma conversa, além de realizar atividades juntos, como jogar mini-jogos (para promover a calma entre usuários). O Usuário PcD não terá acesso a essa funcionalidade.
