## Login e Diagnóstico Diário
- Usuário faz Login e é encaminhado para uma área onde mostra se é necessário a habilitação da acessibilade para o uso especial (Usuário PcD), logo após a análise feita, os usuários (Usuário Comum e Uusário PcD) são direcionados para o Diagnóstico Diário, onde serão coletadas informações de como o usuário está se sentindo.

## Mensagem Anônima
- O Usuário Comum Poderá enviar mensagens anônimas para outros usuários, sem ser identificado sendo mensagens de apoio emocional ou simplemenste um emoji feliz. (Usuário PcD não poderá mandar mensagens anônimas, apenas receber)

## Área Desabafo
- O Usuário Comum terá uma Área do Desabafo onde servirá como um diário, anotando suas frustações e até mesmo acontecimentos do seu cotidiano. (Usuário PcD não possuirá acesso)

## Ombro Amigo
- O Usuário Comum terá acesso a funcinalidade Ombro Amigo, que irá acionar um amigo vinculado para uma simples interação (pedidos de ajuda ou para conversar) e até realizar tarefas juntos como jogar mini-jogos (para promover a calma entre usuários). (Usuário PcD não possuirá acesso)
